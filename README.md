# Acceleration for NVMe Interface IP

The repo is born to accelerate the nvme interface ip from OpenSSD project, 
which concretely is included into daisyplus-nvme project. 

Now it can only support the fw on Daisyplus board in Vitis2024.2

And add the hw from the `OpenSSD-Proj/DaisyPlus/NVMe/LPDDR4/...nvme_interface` 
it can only support pcie3.0 x16, and databitwidth cannot reconf except from 
addrbitwidth.
